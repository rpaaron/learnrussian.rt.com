Remember

When we use the number 1 (один, одна, одно), we match the gender of this number with the gender of the noun it is modifying:

   

он

один

брат, друг, стол, компьютер, билет

она

одна

сестра, подруга, проблема, касса, система, газета

оно

одно

пиво, вино, окно, море

To express complex numbers like 21, 22, etc, first we should name a bigger number and then the smaller one

25 (двадцать пять) — это 20 (двадцать) → 5 (пять).

74 (семьдесят четыре) — это 70 (семьдесят) → 4 (четыре).

169 (сто шестьдесят девять) — это 100 (сто) → 60 (шестьдесят) → 9 (девять)
